# Gestão de Estoque de Veículos

Projeto desenvolvido em **Python** utilizando **Streamlit**, com o objetivo de simular a gestão de estoques veicular.

## Objetivo do Projeto
Este projeto tem como finalidade praticar:
- Programação em Python
- Criação de interfaces web com Streamit
- Organização de projetos
- Versionamento com Git e GitHub

Além disso, o projeto evoluirá para incluir conceitos úteis em:
- Análise de dados
- Automação
- Segurança da Informação
- Cloud e conteiners

## Tecnologias Utilizadas
- Python 3
- Streamlit
- Git e GitHub
